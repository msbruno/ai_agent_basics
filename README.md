# ai_agent_basics
A Basic playground for AI Agents with Ollama

## STEP 1 - Basic AI Agent
1- Install Ollama
(https://ollama.com/download)

2- Create Git repo and clone

3- Add Venv
- ```$ pipx/venvs/virtualenv/bin/python3 -m virtualenv venv```

4- Run Mistral
- ```$ ollama run mistral```   

5- Activate venv
- ```$ source ./venv/bin/activate```

6- Install Requests
- ```$ pip install requests```

7- Run agent
- ```$ python agent.py```
